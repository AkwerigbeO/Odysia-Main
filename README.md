# Odysia Event Management Platform

This is the monorepo for the Odysia Event Management Platform.

## Structure

- **frontend/**: The Next.js frontend application.
- **backend/**: The Express.js backend API.

## Getting Started

### Quick Start (Root)

Run both frontend and backend from the root directory:

```bash
npm install
npm run dev
```

### Frontend

Navigate to the frontend directory:

```bash
cd frontend
npm install
npm run dev
```

### Backend

Navigate to the backend directory:

```bash
cd backend
npm install
npm run dev
```
